# 语音输入同步系统 - 使用手册

> 手机端语音输入 → 实时同步到电脑 → 自动打字到任意窗口

---

## 🎯 功能特点

- ✅ 使用手机语音输入法（豆包、搜狗等）
- ✅ 实时同步到电脑，无需点击发送
- ✅ 支持任意窗口输入（Word、记事本、聊天框等）
- ✅ 拟态风格UI设计，舒适美观
- ✅ 支持文字输入、清空、换行、删除等操作
- ✅ 后台运行，不影响电脑使用
- ✅ 支持开机自启

---

## 📋 系统要求

**电脑端：**
- Windows 10/11
- Python 3.7+
- 网络连接（WiFi或网线）

**手机端：**
- 任意智能手机
- 任意浏览器
- 任意语音输入法（推荐：豆包输入法）

## 🚀 快速开始

### 第一步：安装依赖

```bash
pip install websockets keyboard
```

### 第二步：启动系统

**方法1：一键启动（推荐）**
```bash
双击运行 start.bat
```

**方法2：手动启动**
```bash
# 1. 启动HTTP服务器
python -m http.server 8000

# 2. 启动WebSocket服务器（新窗口）
python server.py

# 3. 启动客户端（新窗口）
python client.py
```

### 第三步：手机连接

1. 确保手机和电脑连接同一WiFi
2. 手机浏览器访问显示的网址（如：`http://192.168.1.100:8000/mobile.html`）
3. 在手机页面输入框使用语音输入法
4. 文字自动同步到电脑

---

## 🎮 使用方法

### 基本使用

1. **在电脑上**：打开任意需要输入的窗口，光标放在输入位置
2. **在手机上**：点击输入框，使用语音输入法说话
3. **自动同步**：文字实时出现在电脑上

### 控制按钮

| 按钮 | 功能 | 说明 |
|------|------|------|
| 🗑️ 清空 | 清空所有文字 | 手机端和电脑端同时清空 |
| ↵ 换行 | 发送回车键 | 换行或确认 |
| 📤 发送全部 | 发送所有文字 | 将输入框中的所有文字发送到电脑 |
| ⌫ 删除 | 发送退格键 | 删除一个字符 |
| ␣ 空格 | 发送空格键 | 输入空格 |
| ⇥ Tab | 发送Tab键 | 切换焦点或缩进 |

## 🔧 故障排查

### 手机无法访问网页

**原因：**
- 手机和电脑不在同一局域网
- 防火墙阻止访问
- 路由器启用AP隔离

**解决方案：**
1. 确保手机和电脑连接同一WiFi
2. 临时关闭防火墙测试
3. 添加防火墙规则：
   ```bash
   netsh advfirewall firewall add rule name="VoiceInputSync" dir=in action=allow protocol=tcp localport=8000
   netsh advfirewall firewall add rule name="VoiceInputSync-WS" dir=in action=allow protocol=tcp localport=8765
   ```

### 文字没有输入

**原因：**
- 客户端未运行
- 输入框未获得焦点
- 权限不足

**解决方案：**
1. 检查 `client.py` 是否在运行
2. 确保电脑端输入框有光标闪烁
3. 以管理员权限运行 `client.py`

### 连接断开

**原因：**
- 网络不稳定
- 服务停止

**解决方案：**
- 系统会自动重连，无需操作
- 如长时间无法连接，重启所有服务

### 显示乱码

**原因：**
- 编码问题

**解决方案：**
- 修改 `client.py` 和 `server.py` 开头添加：
  ```python
  import os
  os.environ['PYTHONIOENCODING'] = 'utf-8'
  ```

## 🌟 推荐配置

### 最佳实践

1. **语音输入法**：推荐使用豆包输入法（识别准确）
2. **手机放置**：手机放在桌面，方便边看边说
3. **窗口布局**：电脑窗口左侧，手机右侧（或分屏）
4. **网络环境**：使用5GHz WiFi，延迟更低

## 📊 技术原理

### 架构流程

```
手机语音输入 → 文字识别
     ↓
WebSocket发送 → 服务器中转
     ↓
电脑客户端接收 → keyboard库模拟输入
     ↓
当前窗口显示文字
```

### 关键技术

- **WebSocket**：实时双向通信
- **keyboard库**：模拟键盘输入
- **HTTP服务器**：提供手机访问页面

## 🤝 常见问题

**Q: 支持哪些语音输入法？**
A: 支持所有手机语音输入法，推荐豆包、搜狗、讯飞等。

**Q: 可以同时连接多台手机吗？**
A: 可以，但文字会混在一起，建议一次一台。

**Q: 会影响电脑性能吗？**
A: 不会，资源占用极低（<10MB内存）。

**Q: 支持Mac/Linux吗？**
A: 目前仅支持Windows，Mac/Linux需要修改keyboard库。

**Q: 可以远程使用吗？**
A: 可以，需要公网IP或内网穿透（如frp、ngrok）。
