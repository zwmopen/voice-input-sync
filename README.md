## 🚀 快速开始

### 1. 安装依赖

```bash
pip install websockets keyboard
```

### 2. 启动系统

```bash
双击运行 start.bat
```

或手动启动：

```bash
# 启动HTTP服务器
python -m http.server 8000

# 启动WebSocket服务器
python server.py

# 启动客户端
python client.py
```

### 3. 手机连接

**重要说明：手机无法通过电脑名访问，请使用IP地址！**

1. 手机和电脑连接同一WiFi
2. 查看启动脚本显示的IP地址（如：`http://10.214.31.7:8000/mobile.html`）
3. 手机浏览器访问该地址
4. 使用语音输入法开始输入

**访问地址说明：**
- ✅ **推荐**：使用IP地址（如 `http://10.214.31.7:8000/mobile.html`）
- ❌ **不可用**：电脑名地址（`http://zwm:8000/mobile.html`）在手机上无法访问

## 📖 使用方法

1. **电脑端**：打开任意窗口，光标放在输入位置
2. **手机端**：点击输入框，使用语音输入法说话
3. **自动同步**：文字实时出现在电脑上

### 控制按钮

| 按钮 | 功能 |
|------|------|
| 🗑️ 清空 | 清空手机和电脑端所有文字（智能计算删除时长） |
| ↵ 换行 | 发送回车键 |
| 📤 发送全部 | 发送输入框中所有文字 |
| ⌫ 删除 | 发送退格键 |
| ␣ 空格 | 发送空格键 |
| ⇥ Tab | 发送Tab键 |

### 手机键盘支持

- ✅ **删除键**：手机键盘的删除键会同步到电脑
- ✅ **回车键**：手机键盘的回车键会同步到电脑
- ✅ **实时统计**：显示总字数、本次字数、清空时长

### 智能清空功能

系统会自动计算清空需要的时长：

```
清空时长 = 字数 × 12ms × 1.2（加20%余量）
```

例如：
- 100个字 ≈ 1.4秒
- 500个字 ≈ 7秒
- 1000个字 ≈ 14秒

## 🔧 故障排查

### 手机无法访问

**原因：** 手机无法通过电脑名（如 `zwm`）访问电脑

**解决方案：**
1. 使用IP地址访问（如 `http://10.214.31.7:8000/mobile.html`）
2. 确保手机和电脑在同一WiFi
3. 查看启动脚本显示的IP地址
4. 临时关闭防火墙测试

**获取IP地址：**
```bash
# Windows
ipconfig

# 查找 IPv4 地址（排除 169.254.x.x）
```

### 文字没有输入

**原因：**
- 客户端未运行
- 输入框未获得焦点
- 权限不足

**解决方案：**
1. 检查 `client.py` 是否在运行
2. 确保电脑端输入框有光标闪烁
3. 以管理员权限运行 `client.py`

### 连接断开

**原因：**
- 网络不稳定
- 服务停止

**解决方案：**
- 系统会自动重连，无需操作
- 如长时间无法连接，重启所有服务

### 显示乱码

**原因：**
- 编码问题

**解决方案：**
- 修改 `client.py` 和 `server.py` 开头添加：
  ```python
  import os
  os.environ['PYTHONIOENCODING'] = 'utf-8'
  ```

## 📊 技术原理

### 架构流程

```
手机语音输入 → 文字识别
     ↓
WebSocket发送 → 服务器中转
     ↓
电脑客户端接收 → keyboard库模拟输入
     ↓
当前窗口显示文字
```

### 关键技术

- **WebSocket**：实时双向通信
- **keyboard库**：模拟键盘输入
- **HTTP服务器**：提供手机访问页面
- **智能清空**：根据字数计算删除时长，兼容所有场景

## 🤝 常见问题

**Q: 支持哪些语音输入法？**
A: 支持所有手机语音输入法，推荐豆包、搜狗、讯飞等。

**Q: 可以同时连接多台手机吗？**
A: 可以，但文字会混在一起，建议一次一台。

**Q: 会影响电脑性能吗？**
A: 不会，资源占用极低（<10MB内存）。

**Q: 支持Mac/Linux吗？**
A: 目前仅支持Windows，Mac/Linux需要修改keyboard库。

**Q: 可以远程使用吗？**
A: 可以，需要公网IP或内网穿透（如frp、ngrok）。

**Q: 为什么手机无法通过电脑名访问？**
A: 手机DNS无法解析电脑名，请使用IP地址访问。
